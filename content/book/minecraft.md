:text-title{:t="第一次玩 Minecraft"}

还记得我第一次玩 Minecraft 的时候，还是在小学，同学来我家帮我注册邮箱. 记得当时有很多启动器, 里面有很多服务器可以选择. 我影响比较深的就是多玩我的世界盒子和麦块(好像是叫这个名字). 到之后加 QQ 群下载启动器 Hello Minecraft! Launcher(那个时候还不知道是这个) 我影响最深的版本就是 1.7.10 玩了很久这个版本, 同时也进到了很多好玩的生存服务器. 记得那时候最幸福的就是买地皮, 和邻居互相认识. 

那时还不知什么是 RPG, 只知道很好玩, 可以刷金币兑换比较厉害的武器. 玩小服务器的时候服主叫 "腐竹", 有管理权限的叫 op, 五块钱六块就能买个 op 体验创世主的快乐. 不过我没有买, 白嫖到了一个 op 玩. 我觉得 Minecraft 这个游戏很注重引导, 例如那时我根本不知道 Java 是什么, 只知道玩 Minecraft 需要安装 Java. 印象最深的就是 Java 那个进度条和那个红色的背景.

十个好多年再次玩 Minecraft 才发现启动时那个铁砧 Logo 是 [forge](https://files.minecraftforge.net/net/minecraftforge/forge/), 光影能接触到 shader, 这个还是我最近混进类似游戏上下游才知道的. 例如 unity shader. 而材质\ Mod 能接触到 Java 和 Minecraft Forge

毕竟地域和末地去过了, 也打完末影龙了, 觉得一个人玩没有意思了. 于是就加入到某个服务器和别人一起玩. 但是有些服务器吃相太难看了, 于是就开始琢磨联机. 我那时比较流行花生壳进行内网穿透(工作后发现向日葵也是他们家的). 或者用游侠
